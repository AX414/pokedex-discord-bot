Como desenvolver um Bot do Discord
com o uso de Python?

Premissa:

O Curso a seguir tem como intuito permitir com que os ouvintes possam desenvolver seus próprios bots para a plataforma do discord com o uso de audio. Além disso, noções básicas da linguagem python serão ministradas.

Aqui você verá toda a sequência de passos necessários para o desenvolvimento de um bot desde o início do projeto até a execução da aplicação. É esoerado que conhecimento todos tenham a capacidade de: Compreender o entendimento da lógica de programação realizada por trás dos bots da plataforma e recriar os passos ministrados durante o curso para qualquer ideia posterior a ele.

Antes de tudo, para realizar o desenvolvimento precisamos deixar bem claro uma única coisa, não há restrições para o desenvolvi Inicialmente vamos precisar instalar os módulos necessários para o python
